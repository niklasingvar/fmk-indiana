# PROMOTE, NEVER FORK

## Definition
- Knowledge moves up the cone as it proves out: inbox → learning → rule → invariant → purpose.

## Rules
- Promotion moves the fact; the source is deleted and replaced by a link, never copied.
- Agents promote into drafts; humans ratify drafts into active.

## Test
- The same insight alive in two layers means a promotion forked — merge and link.

## Incorporation
- This repo: the promotion pipeline in [docs/context-model/CONTEXT-MODEL.md](../docs/context-model/CONTEXT-MODEL.md) §7.
- System prompt: consolidation and lint prompts drive promotion; the write-back protocol in the system prompt feeds the inbox it promotes from.
