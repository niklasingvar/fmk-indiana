# PRESENTATION RULES

- Scaffold from a template: pre-built slide components, content separated from design (INV-7).
- Rough drafts are token-cheap by design: copy the template, fill content files, never restyle in the first pass.
- Content iterations touch content files only; design iterations touch design files only; a loop that needs both is two loops.
- One idea per slide; the slide title carries the idea, the body carries the evidence.
